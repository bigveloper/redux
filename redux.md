<h1>redux</h1>
1. redux 에서 사용되는 키워드 숙지하기

-   action (액션) : state에 어떠한 변화가 필요하게 될 때 action 을 발생시키게 되고, 하나의 object 로 표현 된다.  
    `{type : "ADD_TODO"}`

    -   action object 는 `type 필드를 필수 적으로 가지고 있어야 하고 그 외의 값은 마음대로 넣어줄 수 있다.

-   reducer (리듀서)

-   store (스토어)

-   dispatch (디스패치)

-   subscribe (구독)
